# Treści zadań na zajęcia

Zaprojektuj graficzny interface użytkownika do programu umożliwiającego tworzenie paragonów do zakupów.
Aplikacja powinna składać się z okna głównego i okna dialogowego.

## Okno główne
Okno powinno posiadać:
    - Menu aplikacji z przyciskiem "Nowy paragon".
    - Kontrolkę zawierającą listę rekordów - paragonów dodanych do tej pory.
    - Kontrolka wyświetlającą tabelkę ze szczegółami paragonu

Po kliknięciu danego rekordu - paragonu z listy, w tabelce ze szczegółami paragonu powinny wyświetlić się itemy z podświetlonego paragonu.

Przycisk 'Nowy paragon' z menu głównego powinien utworzyć nowe okno dialogowe w którym będziemy wprowadzać szczegóły nowego paragonu.


## Okno dialogowe 
Okno dialogowe będzie posiadać elementy:
    - Pola do wprowadzania danych nowej pozycji: nazwa, jednostka, ilość, cena
    - Tabelka wyświetlająca podsumowanie nowego paragonu
    - Przycisk "Dodaj pozycję", powodujący dodanie nowego rekordu do tabelki podsumowującej.
    - Przycisk "Dodaj paragon", który będzie zapisywał nowy paragon do głównego okna aplikacji
